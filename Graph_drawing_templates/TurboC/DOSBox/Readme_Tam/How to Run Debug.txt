Assume you have placed the DOS based program in C:/DOSBox/programs

1. Download DOSBox from www.dosbox.com	and install
2. type command: mount C C:/DOSBox/programs  	# we now have 'C:/DOSBox/programs' as C drive
3. C: <Enter>  					# to have the C prompt  
4. Now carry on to run the executable programs; type there name and press enter.
